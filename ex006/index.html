Principais formatações

Main formats

Negrito / Destaque

Bold / Highlight

Nesta frase temos um termo em negrito usando a tag B (não semântica).

In this sentence we have a term in bold using the tag B (not semantics).

Nesta frase há um termo em destaque usando a tag STRONG (semântica).

In this sentence there is a highlighted term using the tag STRONG (semantics).

Itálico / Ênfase

Italics / Emphasis

Nesta frase temos um termo em itálico usando a tag I (não semântica).

In this sentence we have an italicized term using the I tag (not semantics).

Nesta frase temos um termo em ênfase usando a tag EM (semântica).

In this sentence we have a term in emphasis using the tag EM (semantics).

Texto marcado

Marked text

Podemos criar também um texto marcado usando a tag MARK.

We can also create a marked text using the MARK tag.

E no outro parágrafo, temos outro texto marcado no final.

And in the other paragraph, we have another text marked at the end.

Texto grande e pequeno

Large and small text

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

We are creating large text and small text in that paragraph.

Texto deletado

Deleted text

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

We can mark a text as deleted to indicate that it should be read, but not considered.

Texto inserido

Inserted text

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

We can mark a text as inserted for emphasis and to indicate that it was added later.

Existe também o sublinhado com a tag U (não semântica).

There is also the underline with the U tag (not semantic).

Texto sobrescrito

Superscript text

Para inserir coisas do tipo x20+3

To insert things like x 20 +3

Texto subscrito

Subscribed text

Para inserir coisas do tipo H2O

To insert things like H 2 O
